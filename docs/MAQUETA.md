# Maqueta del MVP

## Pantalla principal

La primera pantalla presenta la identidad de AMARU KIDS, el lema del proyecto y el impacto social que justifica la solucion.

Elementos principales:

- Titulo AMARU KIDS.
- Lema: "La tecnologia tambien debe sentirse, escucharse y comprenderse."
- Estado de servicios: Firebase y Gemini.
- Acceso al prototipo interactivo.

## Perfil de accesibilidad

El usuario puede seleccionar sus necesidades:

- Discapacidad visual.
- Discapacidad auditiva.
- Dificultades de comunicacion.
- TDAH.
- Sin necesidades especificas.

Segun la seleccion, la interfaz activa apoyos como alto contraste, texto ampliado, pictogramas, alertas visuales o actividades cortas.

## Historia interactiva

El MVP incluye la historia "El viaje de Amaru". Cada pagina contiene:

- Texto educativo.
- Pictogramas.
- Boton para escuchar la historia.
- Boton para adaptar el contenido con IA.
- Pregunta de comprension.

## Recompensas

Al responder correctamente, el usuario gana estrellas. Esto mantiene la motivacion y permite medir avance.

## Panel familiar

El panel familiar muestra:

- Historias completadas.
- Tiempo de uso.
- Estrellas obtenidas.

En una version futura, estos datos se guardaran en Firebase y se convertiran en reportes para familias y docentes.
