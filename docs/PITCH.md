# Pitch de presentacion: AMARU KIDS

## 1. Apertura

En Ecuador, miles de ninos enfrentan barreras para aprender con tecnologia porque muchas plataformas no consideran discapacidades visuales, auditivas, dificultades de comunicacion o necesidades educativas especificas.

AMARU KIDS nace para cambiar eso.

## 2. Problema

Las aplicaciones educativas tradicionales suelen ser costosas, estar en otros idiomas, atender una sola necesidad de accesibilidad o no estar pensadas para ninos.

Esto limita la autonomia, la inclusion y las oportunidades de aprendizaje.

## 3. Solucion

AMARU KIDS es una plataforma educativa inclusiva con inteligencia artificial adaptativa. La app ajusta la experiencia segun las necesidades del usuario mediante lectura por voz, pictogramas, alto contraste, actividades interactivas y recompensas.

## 4. Demo del MVP

El prototipo permite:

- Crear un perfil de accesibilidad.
- Leer una historia interactiva.
- Escuchar el contenido por voz.
- Adaptar texto con IA.
- Responder una actividad.
- Ganar estrellas.
- Ver progreso en un panel familiar.

## 5. Diferenciador

No buscamos que los ninos se adapten a la tecnologia. Buscamos que la tecnologia se adapte a los ninos.

## 6. Tecnologia

El MVP esta construido con React + Vite. La arquitectura esta preparada para Firebase como base de datos y Gemini AI como motor de adaptacion inteligente.

## 7. Cierre

AMARU KIDS no es solo una aplicacion educativa. Es una herramienta de inclusion que busca que ningun nino quede fuera del aprendizaje digital.
